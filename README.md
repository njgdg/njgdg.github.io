## Usage
Create note in directory "_posts", name format : 2015-11-11-xxx-xxx-xxx.markdown, note file header sample:

```
---
layout:     post
title:      "Unix/Linux XXX XXX"
subtitle:   "xxx xxx，xxx xxx"
date:       2015-04-14 
author:     "Jerry"
header-img: "img/in-post/post-bg-xxx-xxx.jpg"
tags:
    - OS
    - Unix
    - Linux
---

```
